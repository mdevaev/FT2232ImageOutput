# FT2232ImageOutput
Drawing images on oscilloscope using ft2232hc chip breakout board, 74hc595 shift registers and 8-bit R2R DACs

Executable running example:
```
FT2232ImageOutput.exe -f ..\..\samplefiles\lain.ild
```
